# 💹My Finances

Aplicativo para calcular entradas e saídas financeiras.

## 🚀 Instalação

Clone o projeto

```bash
  git clone https://github.com/quinamilena/weather-app.git
```

Entre no diretório do projeto

```bash
  cd my-finances
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
